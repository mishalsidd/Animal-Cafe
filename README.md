# Animal-Cafe
Python game that allows players to customize their experience by selecting a difficulty level (easy, medium, hard) and choosing from various characters (cat, dog, monkey, bunny). Players complete a series of cake orders, with performance evaluated based on accuracy, resulting in a win or loss.
